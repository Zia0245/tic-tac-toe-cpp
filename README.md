# Tic Tac Toe (C++)

A console-based two-player Tic Tac Toe game implemented in C++.

## Features
- Two-player mode (X and O)
- Input validation
- Win and draw detection
- Clear game board display

## Concepts Used
- 2D arrays
- Functions
- Loops and conditions

## How to Run
Compile using any C++ compiler and run the program.
